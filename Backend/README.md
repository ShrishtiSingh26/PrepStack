# PrepStack Backend

This folder contains the Node.js/Express backend API codebase for PrepStack. 

For full setup instructions, technology overview, and workflow guides, please refer to the main [project README at the root](file:///c:/Users/hp/OneDrive/Documents/GitHub/PrepStack/README.md).

## Quick Start
```bash
npm install
npm run dev
```
Runs the Express API server on `http://localhost:3000`.
